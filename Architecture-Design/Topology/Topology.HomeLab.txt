    CSS610 (Core Switch)
       ↓
    Proxmox + AP + Clients
       ↓
    Services isolated by VLANs
    
    
    
    =========================
     BEST PRACTICES
    =========================
    
    - Keep routing/firewall on RB5009
    - Keep applications inside Proxmox
    - Use Docker only for lightweight services
    - Use dedicated VM for Home Assistant
    - Use dedicated VM for Jellyfin
    - Use dedicated VM for IDS/logging
    - Backup Proxmox nightly
    - Use VLAN isolation everywhere
    - Expose services only via Nginx Proxy Manager
    - Use WireGuard for remote administration
